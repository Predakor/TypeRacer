# Typer

A typing speed app that lets you test your typing speed.

## Demo

[Check Typer online](https://predakor.github.io/typer)

## Installation

Clone the project

```bash
  git clone https://github.com/Predakor/Typer
```

Go to the project directory

```bash
  cd typer
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

## Technologies
